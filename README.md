# Todo List
Uma aplicação simples para a criação de uma lista de tarefas.

# Tecnologias usadas:
- HTML
- -CSS

# Instalação
- Clone o repositório: "git clone https://github.com/ThiagoMarchini/todo-list.git"
- Entre no diretório "cd todo-list"
- Abra o arquivo "index.html"
